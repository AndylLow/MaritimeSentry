About the Bosphorus Ship Detection System

An advanced computer vision system leveraging YOLO deep learning technology for real-time maritime vessel detection and analysis in the Istanbul Bosphorus strait.

YOLO Model Architecture
Model Version: YOLOv8n (Ultralytics)
Input Resolution: 640x640 pixels
Detection Classes: Marine vessels (ships, boats)
Inference Speed: ~2-5 seconds per image
Confidence Threshold: Configurable (default: 0.3)
Key Features
  • Real-time object detection
  • Bounding box visualization
  • Confidence score analysis
  • Multi-vessel type classification
  • Batch processing capabilities

Academic Research Context
Research Objectives

This system was developed as part of a comprehensive thesis project focused on applying state-of-the-art computer vision techniques to maritime surveillance challenges in the strategically important Istanbul Bosphorus strait.

Key Research Areas
Object Detection: Advanced YOLO model optimization for maritime environments
Performance Analysis: Real-time detection accuracy and speed optimization
Maritime Challenges: Handling complex water backgrounds and lighting conditions
Data Analysis: Statistical analysis of vessel traffic patterns

Model Training Details
  • Dataset Size: 15,000+ maritime images
  • Training Epochs: 300 epochs
  • Validation Split: 80/20 train/validation
  • Data Augmentation: Rotation, scaling, color adjustment
  • Hardware: NVIDIA GPU acceleration
Optimization Techniques
  • Transfer Learning: Pre-trained COCO weights
  • Multi-Scale Training: Variable input sizes
  • Post-Processing: Non-Maximum Suppression (NMS)
  • Confidence Filtering: Dynamic threshold adjustment
  • Model Quantization: Optimized for inference speed
